## 📖 Overview

**Resume Roaster** is a premium, AI-powered tool designed for developers who take their careers seriously but can handle a little roasting. Using **Google Gemini**, it brutally analyzes your resume, gives you a "Burn Score," and then provides an ATS-optimized professional version to help you land your dream job.

---

## ✨ Key Features

- **🔥 Brutal AI Roast**: Get a cynical, brutally honest review of your resume's failures.
- **📈 Burn Score**: A 0-100 metric on how much your resume needs fixing.
- **✨ One-Click Rebuild**: Instantly transform your "garbage" resume into a professional, ATS-friendly masterpiece.
- **📄 PDF Support**: Direct text extraction from your PDF files.
- **✏️ Optimization Center**: Fine-tune the AI-generated resume before exporting.
- **💾 Professional Export**: Download your new resume as a clean, formatted PDF.

## 🚀 Getting Started

<details>
<summary><b>1. Clone the repository</b></summary>

```bash
git clone https://github.com/atharvwalawalkar05-pixel/Resume-Roaster.git
cd Resume-Roaster
```
</details>

<details>
<summary><b>2. Install Dependencies</b></summary>

Make sure you have [Node.js](https://nodejs.org/) installed, then run:
```bash
npm install
```
</details>

<details>
<summary><b>3. Set up Environment Variables</b></summary>

Create a `.env` file in the root directory and add your [Google Gemini API Key](https://aistudio.google.com/):
```env
VITE_GEMINI_API_KEY=your_actual_api_key_here
```
</details>

<details>
<summary><b>4. Run the Development Server</b></summary>

```bash
npm run dev
```
</details>

<details>
<summary><b>5. Open the App</b></summary>

Navigate to `http://localhost:5173` in your browser.
</details>
